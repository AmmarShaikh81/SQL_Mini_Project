#Customer Book Order Analysis (SQL Project)

## Overview
This project analyzes customer purchasing behavior and book ordering patterns using SQL. It helps identify popular genres, top customers, and sales trends.

---

## Objectives
- Find most popular book genres
- Identify top customers
- Analyze revenue and sales
- Track book demand and stock

---

## SQL Concepts Used
- Joins (INNER, LEFT)
- GROUP BY, ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- Subqueries
- COALESCE


##Advanced SQL:
- CTE (WITH clause)
- Window Functions (RANK, ROW_NUMBER)
